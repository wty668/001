# ETF行情通 - A股ETF行情与投教演示

ETF行情通是一个模拟A股ETF行情展示与投教内容的小程序风格HTML演示页面。

## 功能模块

- **发现页**：Banner轮播、A股指数、ETF市场表现（8分类）、热门ETF指数、投教精选
- **ETF筛选**：搜索、分类筛选（宽基/行业/主题/T+0/跨境/商品/债券/货币）、子分类、排序
- **ETF投教**：学堂Banner、分类内容（文章/视频）、快速入门、FAQ

## 技术栈

- 纯HTML + CSS + JavaScript（单文件，无依赖）
- JSON数据驱动（支持后台更新）
- 红色金融主题配色

## 本地预览

直接用浏览器打开 `etf-miniprogram-demo.html` 即可。

## 部署

支持部署到 Vercel、GitHub Pages 等静态托管平台。
